# Claude AI Project

Welcome to the **Claude_ai** repository! This project contains scripts, workflows, and experiments built using Anthropic's Claude API.

---

## 📌 Features

* **API Integration:** Simple setup and usage examples for Claude models.
* **Prompt Engineering:** Practical prompt templates and structured output handling.
* **Modular Code:** Organized daily progress and project modules (e.g., `day 7`).
